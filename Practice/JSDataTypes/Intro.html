Data Types in JS

In javascript there are two main types of datatypes.

1.Premitive Data Type
2.Non-Premitive Data Type

Types of premitive data type

1.Number

let a = 10;
console.log(a);
console.log(typeof a);

2.Null

let b = null;
console.log(b);
console.log(typeof b);

3.String

let c = 'Tarun'
let c = "Tarun"
let c = "Hey I'm a good Boy"
let c = `Hey I'm a good "Boiiii"`; // BackTick i.e Template Literals
console.log(c);
console.log(typeof c);

4.Symbol

let d = Symbol("Rahul")
console.log(d);
console.log(typeof d);

5.BigInt

let e = 561156454512561654614546n
console.log(e);
console.log(typeof e);

6.Boolean

let f = false
console.log(f);
console.log(typeof f);

7.Undefined

let g;
console.log(g); // undefined
console.log(h); // Not defined
console.log(typeof g);

Types of non-premitive data type

1.Array

let marks = [10,20,'Mansi',40,50]
console.log(marks);
console.log(marks.length);
console.log(marks[8]);
console.log(typeof marks);

2.Object

let details = {
name: 'Arma',
age: 21,
goodStudent: true,
}
console.log(details);
console.log(details.name);
console.log(details.age);
console.log(typeof details);
console.log(typeof details.name);
