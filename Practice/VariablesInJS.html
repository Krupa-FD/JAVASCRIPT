Variables in JS

In JavaScript, there are two main types of variables and one constant.

1.var
2.let
3. const

1.var:

var is the original way to declare variables in JavaScript. It has function scope, which means it is accessible anywhere within the function it was declared in, even before the actual declaration. If a variable is declared outside of any function, it becomes a global variable.

var a;
var a = 10; // Initialization and Declaration or Asignemnt
var a = 15; // Re-Declaration is possible
a = 25; // Re-Assignment is possible
console.log(a);

2.let:

let allows you to declare block-scoped variables. This means the variable is only accessible within the block (a block is a section of code between curly braces {}), statement, or expression where it is defined.

let nirav_marks = 90 // Initialization and Declaration or Asignemnt
let nirav_marks = 40 // Re-Declaration is not possible
nirav_marks = 75 // Re-assignemnt Possible
console.log(nirav_marks);

3.const:

const is used to declare variables that cannot be reassigned after their initial assignment. Like let, const is block-scoped. It's important to note that while a const variable cannot be reassigned, if it holds an object or an array, the properties or elements of that object or array can still be modified.

const fav_Person = "nirav"; // Initialization and Declaration or Asignemnt
const fav_Person = "krina" // Re-Declaration is not possible
fav_Person = "krupa" // Re-Assignemnt is not possible
console.log(fav_Person);
